## Dashboard Features

The dashboard provides five interactive views to analyze player value:

1. **Executive Overview** - League-wide salary and performance metrics with interactive filters
2. **Salary Distribution** - Visual breakdown of how money is distributed across players
3. **Performance vs Salary** - Scatter plot revealing the correlation (or lack thereof) between pay and production
4. **Team Payroll Comparison** - Compare how all 30 teams spend their money and perform
5. **Trade & Free Agency Intel** - Identify which players to sign, trade away, pursue, or watch

## What This Project Does

This project builds a complete data pipeline that:

1. **Extracts** player statistics from the NBA API and Basketball Reference (web scraping)
2. **Extracts** salary data from HoopsHype (web scraping)
3. **Stores** all data in a MySQL database
4. **Transforms** raw data using Python scripts to calculate efficiency metrics
5. **Exports** clean CSV files for Tableau visualization
6. **Visualizes** insights through an interactive dashboard

## Data Sources

### Player Statistics
- **NBA API** (`nba_api` Python library) - Basic stats (PPG, RPG, APG, shooting percentages)
- **Basketball Reference** (web scraping) - Advanced metrics (VORP, Win Shares, BPM, PER)

### Salary Data
- **HoopsHype** (web scraping) - Player salaries for 2022-23, 2023-24, and 2024-25 seasons

## Key Insights

What the dashboard reveals:

- **Salary ‚â† Performance**: Only 35-45% of performance variance is explained by salary
- **Hidden Gems Exist**: Many low-salary players ($5-12M) deliver exceptional value
- **Overpaid Players**: The $18-25M salary range often yields the worst value
- **Elite Players Are Worth It**: Top performers with max contracts justify their pay
- **Position Economics**: Combo guards and stretch bigs provide best value; traditional centers are often overpaid

## Calculated Metrics

The Python scripts calculate value metrics including:

- **Points per Million** = PPG / (Salary / 1,000,000)
- **Salary per Win Share** = Salary / Win Shares
- **Salary per VORP** = Salary / Value Over Replacement
- **Value Score** = (VORP √ó 1,000,000) / Salary √ó 100
- **Value Categories** = "Hidden Gem", "Good Value", "Overpaid", etc.
